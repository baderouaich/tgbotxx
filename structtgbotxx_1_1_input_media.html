Detailed Description

This object represents the content of a media message to be sent. It should be one of:

  • InputMediaAnimation
  • InputMediaDocument
  • InputMediaAudio
  • InputMediaPhoto
  • InputMediaVideo https://core.telegram.org/bots/api#inputmedia
